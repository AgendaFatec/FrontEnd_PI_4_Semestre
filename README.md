🛠️ Como executar o projeto
Siga os passos abaixo para rodar a aplicação na sua máquina:

1. Clonar o repositório
Abra o terminal e execute:

Bash
git clone [LINK_DO_SEU_REPOSITORIO]
2. Instalar as dependências
Entre na pasta do projeto e instale os pacotes necessários (isso criará a sua pasta node_modules local):

Bash
cd FRONTEND_PI_4_SEMESTRE
npm install
3. Rodar o ambiente de desenvolvimento
Para subir o servidor e visualizar o projeto no navegador:

Bash
npm run dev
Após rodar o comando, clique no link que aparecer no terminal (geralmente http://localhost:5173).
